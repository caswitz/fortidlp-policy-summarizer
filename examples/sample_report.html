How to Read This Report

How FortiDLP Policies Work

FortiDLP policies monitor endpoint and user activity for security-relevant events. Each policy defines:

  • What to detect — conditions that identify risky or policy-violating activity
  • How severe it is — risk score and MITRE ATT&CK mapping for triage priority
  • How to respond — automated actions: block, notify, screenshot, or log only

Detection Flow

Policy Groups and Targeting

Policies are organized into groups scoped by labels — e.g., a group labeled "Windows" applies only to matching endpoints. Groups labeled "All systems" have no targeting restriction.

Sensitive file transferred via video conferencing application90 Critical

Detects when a user opens a file containing specified content which was transferred using a video conferencing application, such as WebEx, Skype, or Slack.

Notes:
  • Two sets of parameters must be configured to trigger a detection: (1) at least one sensitivity parameter that specifies either a data identifier (content inspection pattern, keyword/keyphrase, or Microsoft sensitivity label), origin, file type, or file size; (2) at least one context parameter that specifies either a process, file path, or user (non-system account).
  • Subsequent detections and actions for the same file and process will not be generated until at least 10 seconds after the first detection/action.
  • From Agent 9.1.0+, the "File extensions" parameter is deprecated and will only be used if the "File types" parameter (preferred) is not configured.

Requirements: Agent 8.4.0+

Tags: apps datatracking filecopy dlp

Triggers: Triggers when any of these applications are used: Amazon Chime, chime.exe, BlueJeans, bluejeans.exe, Cisco Webex Meetings, CiscoCollabHost.exe, CiscoCollabHostCef.exe, GoToMeeting, g2mcomm.exe, g2mlauncher.exe, g2mstart.exe, join.me, join.me.exe, join.me.sentinel.exe, MyMeetingVideo.exe, Pexip (My Meeting Video), RingCentral Meetings, ringcentralmeetings.exe, Skype, skype.exe, Slack, slack.exe, Teams, teams.exe, Webex Teams, webex.exe, atmgr.exe, zoom.exe, zoom.us

Content inspection:

  • Content patterns: 2 predefined asset(s) referenced
  • Match requirement: Match any
  • Each pattern/keyword must appear at least 3 times to trigger

Exceptions:

  • Exempt file paths: *:\Users\*\Documents\Zoom\**, /home/*/Documents/Zoom/**

Settings:

  • Monitors files opened for reading
  • Detection grouping window: 30 seconds

Customizable: Called paths (allow list), Application window titles (Windows and macOS only) (deny list), File types (allow list), File extensions (allow list), File path keywords (allow list)

Response: Monitor only
